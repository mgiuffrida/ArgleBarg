## Introduction

Say "**=**" as "**e**quals", but **si**lence for **dots**,  
same with **com**mas and **col**ons, and **brack**ets in **spots**.

## Sample Usage

```
ArgleBarg().bargle('arg')
ArgleBarg().fargen(
    ('arg', 'fargen'), ('arg', 'fargen'),
    ('far', 'men', 'far', 'barg'),
    par='bargenfarg', farg='farg')
```
